# 🧪 PharmaZone — The Future of Transparent Healthcare

PharmaZone is a high-end, "transparency-first" pharmaceutical e-commerce and health management platform. Designed with an "Apple-style" premium aesthetic, it empowers users to save up to 70% on healthcare costs through automated molecule matching and live price comparisons across India's top pharmacies.

![PharmaZone Hero](https://images.unsplash.com/photo-1587854692152-cbe660dbbb88?auto=format&fit=crop&q=80&w=1200)

## 🚀 Key Features

### 🔍 Unified Pharma Matrix (Search)
*   **Molecular Deep Scan**: Don't just search for brands; search for the underlying salt.
*   **SmartSwitch™ Substitutes**: Instantly find high-efficacy generic alternatives for expensive brands.
*   **Live Price Comparison**: Compare real-time prices from PharmEasy, Tata 1mg, Netmeds, Apollo, and Blinkit.
*   **Salt Analysis**: Professional molecular breakdowns of over 20+ critical medicine categories including Cancer, Heart, and Diabetes.

### 🛡️ Secure Patient Portal
*   **Smart Rx Upload**: Mandatory prescription verification for Rx-restricted medicines with automated molecule-match scanning.
*   **Digital Rx Vault**: Securely download and manage your past prescriptions in PDF format.
*   **Live Consult**: Direct, low-latency communication with medical specialists for instant clinical reviews.

### 📱 Connected Experience
*   **WhatsApp Share**: Instantly share price comparisons and order summaries with one click.
*   **Price Radar**: Visual tracking of medication costs across the fragment ecosystem.
*   **Knowledge Hub**: Deep medical insights and articles on generic medications and pharmaceutical economics.

## 🎨 Design Aesthetic
*   **Forest Green & Amber Palette**: High-contrast, sterile, and premium visual system.
*   **Glassmorphism & Micro-animations**: Fluid interaction design with deep-scanning UI effects.
*   **Typography**: Powered by *Outfit* and *Inter* for maximum readability and a modern feel.

## 🛠️ Tech Stack
*   **Frontend**: React.js, Tailwind CSS
*   **Icons/Animations**: Lucide React, CSS3 Keyframes
*   **PDF Engine**: jsPDF
*   **State Management**: LocalStorage (Persistence Mode)

# Clone the repository
