### 🌟 Vision
**PharmaZone** is a revolutionary, "transparency-first" healthcare e-commerce platform. We stop users from paying the "Brand Tax" by exposing the underlying molecular composition of medicines and providing real-time data across the fragmented Indian pharmaceutical market.

> *"Sasti Dawai, Asli Kamai. Paisa Vasool, Swasthya Cool."*

## 🛠️ Technology & Architecture

- **Glassmorphism UI**: High-fidelity design system inspired by modern OS aesthetics.
- **Neural Pricing Agent**: Simulates real-time data fetching from competing health tech APIs.
- **WhatsApp Integration**: Instant 1-click sharing of pharmaceutical reports and order receipts.
- **PDF Engine**: Local generation of Digital Prescriptions with authorized medical signatures.

## 📦 Getting Started

```bash
# Clone the repository
git clone https://github.com/RiyaThakur-dotcom/PharmaZone.git

# Enter the directory
cd PharmaZone

# Install dependencies
npm install

# Launch Development Mode
npm run dev
```
